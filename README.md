# Проект автотестов на Pytest

Учебный проект для демонстрации работы с фикстурами в **Pytest**.

## Описание
В проекте реализованы тесты для проверки функционала авторизации. Основной упор сделан на использование иерархии фикстур (`browser` -> `login_page`).

## Стек технологий
* **Python 3.x**
* **Pytest**
